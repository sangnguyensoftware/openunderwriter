The yesorno type supports the entry of true or false values from pick lists. The format options supported by the attribute affect the rendering/validate in the follow ways:

PageFlow 
	<rowScroller [section scroller options]>
		<attributeField title="title" binding="xpath binding" [renderHint="radio|checkbox"] [condition="XPath expression"] [columnCondition="XPath expression"]/>
 	</rowScroller>
  • title Title rendered at the top of the column.
  • binding xpath binding referring to an Attribute in the model to which this column is bound.
  • renderHint The options will be rendered as a list of radio buttons when renderHint="radio" or as a tickbox when renderHint="checkbox" is present. The default is to display the list as a drop down list.
  • condition An optional XPath condition that controls whether a given cell appears in the table or is hidden.
  • columnCondition An optional XPath expression that can be used to show/hide the entire column.
Model 
	<attribute id="id" value="value" format="yesorno"/>
