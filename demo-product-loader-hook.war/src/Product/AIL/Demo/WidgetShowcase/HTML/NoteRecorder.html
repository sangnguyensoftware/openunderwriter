This widget that gives the user the opportunity to record a note.

PageFlow 
<noteRecorder [typeDescriminator="discriminator""]
                 [binding="object binding"]
                 [renderHint="render hint"]/>
  • render hint - Render hint to be included in the widget generated output.
  • typeDiscriminator - Defines a discriminator which refines the types of note that can be added.
Examples

Record a note against the current Policy.

<noteRecorder />

Allow the user to record a note but restrict the note types to 'cancellation reason' types.

<noteRecorder typeDiscriminator="cancellation_reason" />
