The SavedQuotations widget is different from most other widgets in that it doesn't work within the context of a quote, but instead creates its own context of a list of saved quotation. The Widget lists the quotations which the user has previously made but not taken up. Basic information about each quote is displayed including the quotation date and the expiry date and premium. The user is give the option to select a quote as the basis of a requote, to confirm and pay a quote, or to view (download) the quotation document.

        <savedQuotations [requoteDestinationPageId="requote page"] 
                            [confirmAndPayDestinationPageId="pay page id"] 
                            [requoteDisabled="requote disabled"] 
                            [confirmAndPayDisabled="confirm and pay disabled"] 
                            [viewQuotationDisabled="view quotation disabled"] />
  • requote page Page ID of the page to be opened if the user selects the requote option.
  • pay page id The ID of the page to be opened if the user selects the confirm and pay option.
  • requote disabled Set to "true" to hide the requote button.
  • confirm and pay disabled Set to "true" to hide the confirm and pay option.
  • view quotation disabled set to "true" to hide the view quotation button.
