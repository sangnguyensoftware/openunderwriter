A question / command action widget that forwards the page flow to a subsequent question page if the user makes a specific selection.

This widget handles a situation where a form requires further questions of any type depending on the answer given to a question. A question asking for trading status for instance could prompt for additional details about the directors.

Pageflow 

	<question title="Choose Red to go to extended questions page" binding="/asset[id='questionAssetData']/attribute[id='choice']" />
	<commandButtonAction id="submit" label="Submit" destinationPageId="QuestionWithExtendedDetails" styleClass="submit" />
	<pageForwardAction when="onProcessActions" condition="i:test(asset[id='questionAssetData']/attribute[id='choice']/value[.='Red'])" destinationPageId="RedChoicePage"  />
	<commandButtonAction />
  • question - binding The value that the conditional test is assessed on.
  • commandButtonAction - destinationPageId Default page destination.
  • pageForwardAction - destinationPageId Destination page if specific choice is selected.
  • condition Page selector.
  • when When the action is processed.
