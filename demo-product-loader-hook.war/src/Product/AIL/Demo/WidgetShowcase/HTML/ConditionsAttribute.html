In this case the answer to the "Display conditional section" question is bound to the "section" attribute. A condition has then been applied to the parsedUrlContent widget so that it will only display its output on condition that the attribute's value is "Yes". The parsedUrlContent widget has been used in this example, but the condition attribute is available on all widget types.

Note that this condition is applied when the page is loaded. Use the Submit Page button at the foot of the page to force the page to reload.

PageFlow 
<question title="Display conditional section" binding="/asset[id='conditionsData']/attribute[id='section']" />
<parsedUrlContent url="~/HTML/ConditionMet.html" condition="i:test(/asset[id='conditionsData']/attribute[id='section']/value[.='Yes'])"/>
Model 
<asset id="conditionsData" assetTypeId="ConditionsData">
    <attribute id="section" value="No" format="yesorno" />
</asset>
