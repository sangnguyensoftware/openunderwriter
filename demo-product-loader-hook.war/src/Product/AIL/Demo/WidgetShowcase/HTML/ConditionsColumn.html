An attribute field may be included or excluded from a row scroller (or section scroller) using the columnCondition attribute. In the example here a condition has been applied to the Column 2. If the question "Display conditional column" is answers "Yes" then column will be included; if not, it will be excluded.

Like all other widgets, attributeField also supports the condition attribute. This can be used to excluded/included specific cells in a row scroller, but does not affect whole column. It is also possible to exclude whole rows by manipulating the row scroller's binding expression.

Note that this condition is applied when the page is loaded. Use the Submit Page button at the foot of the page to force the page to reload.

PageFlow 
<question title="Display conditional column" binding="/asset[id='conditionsData']/attribute[id='column']" />
<rowScroller binding="/asset[id='conditionsColumnData']" type="ConditionsColumnData" addAndDeleteEnabled="false">
    <attributeField title="Column 1" binding="attribute[id='string1']" />
    <attributeField title="Column 2" subTitle="(conditional)" binding="attribute[id='string2']" columnCondition="i:test(/asset[id='conditionsData']/attribute[id='column']/value[.='Yes'])" />
    <attributeField title="Column 3" binding="attribute[id='string3']" />
    <attributeField title="Column 4" binding="attribute[id='string4']" />
</rowScroller>
