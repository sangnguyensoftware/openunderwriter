Pages in PageFlow can one of two types: QuestionPage, as describer here, or InformationPage. An Information page is one that contains only readable information and navigation widgets. Whereas, a QuestionPage is one that support all of the form fields and invites user input.

    <questionPage title="title" [stage="stage"] [showPageError="showPageError"]/>
  • title Title text to appear at the head of the page.
  • stage Name of the stage that this page is part of. This value is used to group pages into the progress bar. For example, if four pages all define their stage as "Payment" then the progress bar will include a single "Payment" stage and that stage will be shown as the current stage whenever one of those four pages is being viewed.
  • showPageError Can be 'true' or 'false'. If set to 'true' an error banner will be displayed on the page when any errors are present. Defaults to 'true'.

A progress bar will be included at the top of the page when two conditions are met:

  1. The PageFlow's progressBar attribute is set to true.
  2. The page defines a stage attribute.

This gives the PageFlow author the freedom to hide the progress bar on some pages (by leaving state undefined), or to disable it altogether by setting the PageFlow's progressBar attribute to false (which is the default setting).
